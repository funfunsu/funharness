一款 AI 驱动的研发流程管理工具，集成多智能体协作能力，为开发者提供从需求到代码的全流程自动化支持，适配 VS Code 开发环境，让研发效率翻倍。
✨ 核心功能
提供研发全流程闭环支持，覆盖需求、设计、测试、任务、代码全环节，无需切换工具，在 VS Code 内完成一站式研发：
- 需求文档生成：根据用户输入的原始需求，自动生成规范、完整的 PRD（产品需求文档），规避需求模糊、遗漏问题；
- 设计文档生成：基于 PRD 自动输出技术设计文档，明确技术选型、架构设计、接口定义等核心内容；
- 测试用例与脚本生成（可选）：根据设计文档，自动生成适配业务场景的测试用例，支持生成可执行测试脚本；
- 任务拆分：将整体研发任务自动拆分为可执行的细粒度任务，明确任务优先级和依赖关系；
- 代码生成指引：结合拆分后的任务，输出精准的代码生成指引，适配多种开发语言，降低编码成本。
🤖 智能驱动支持
集成多 AI 模型，摆脱单一模型限制，确保输出稳定、精准：
- 支持 GitHub Copilot 协同编码，提升代码生成效率；
- 兼容 Claude CLI，拓展 AI 能力边界，适配不同场景下的生成需求。
🔧 解决的核心问题
- AI 输出不稳定：多模型协同+标准化流程约束，确保需求、设计、代码等输出内容规范、一致，避免无效生成；
- 提升AI Coding效率：多个智能体并行工作，同时处理需求分析、设计、测试等任务，缩短研发周期；
- 任务管理混乱：任务拆分可视化，清晰展示任务优先级、依赖关系，便于跟踪研发进度，避免遗漏。
